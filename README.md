# subsidium

A team support time tracker.

Node, React, Redux, MongoDB and others.

## setup

```
$ npm install
$ sudo /etc/init.d/mongod restart
$ npm run dev
```

browse to http://localhost:8888

## tests

```
$ npm run test
```
